# ReqClarity AI

A modern, web-based tool for automated Software Requirements Specification (SRS) quality analysis using rule-based NLP and AI-powered suggestions.

## Project Structure
- `/client` — React 18 frontend (Vite)
- `/server` — Node.js + Express backend

## Getting Started

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

## Tech Stack
- Frontend: React 18, CSS, Recharts, shadcn/ui
- Backend: Node.js, Express, MongoDB, pdf-parse, compromise
- AI: Claude API (Anthropic), OpenAI API
